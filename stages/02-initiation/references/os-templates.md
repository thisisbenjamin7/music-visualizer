# OS Template Reference

Layout patterns for OS-themed UI. Used in stages 02 and 03.

## Selected Template: {{OS_TEMPLATE}}

---

## Windows 98

### Window Chrome Elements
- Title bar: gradient from #000080 to #1084d0, white text, close/minimize/maximize buttons
- Menu bar: flat grey #c0c0c0 with File, View, Options menus
- Status bar: sunken border, shows current track name and artist
- Body: white or light grey canvas area for the visualizer

### Font
- System font stack: "MS Sans Serif", "Tahoma", sans-serif, 12px

### Key CSS Patterns
- Borders: 2px outset for raised elements, 2px inset for sunken
- Background: #c0c0c0 (system grey)

---

## Mac Classic (System 7)

### Window Chrome Elements
- Title bar: horizontal pinstripe pattern, close box (square) top left, centered title text
- Menu bar: flat white at top of screen, Apple menu, File, Options
- Body: white canvas area

### Font
- Chicago or Geneva equivalent: "Geneva", "Helvetica", sans-serif, 12px

### Key CSS Patterns
- Borders: 1px solid black
- Background: white

---

## Windows XP

### Window Chrome Elements
- Title bar: rounded top corners, blue Luna gradient, white title text, Luna-style close/min/max buttons
- Taskbar: blue bar at bottom with Start button
- Body: white or silver (#ece9d8) canvas

### Font
- "Tahoma", sans-serif, 11px

### Key CSS Patterns
- Title bar gradient: linear-gradient(#0a246a, #a6b5da)
- Button radius: 3px
